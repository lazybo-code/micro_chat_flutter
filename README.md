# flutter 聊天工具第一版

### 功能
- [x] 登录
- [x] 注册
- [ ] 查找好友
- [ ] 添加好友
- [x] 更新好友申请
- [x] 发送文本消息
- [ ] 发送图片消息
- [ ] 发送语音消息
- [ ] 设置功能
- [ ] 厂商推送
- [ ] 消息语音提示
- [ ] 数据本地化
